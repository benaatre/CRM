# CRM Schema — أعمدة وقيم قاعدة العملاء

The **Leads** sheet has these columns, in this order (right-to-left layout, so
column A appears on the far right in Excel Arabic view).

| المفتاح (internal)   | العمود (header)        | الوصف                                            |
|----------------------|------------------------|--------------------------------------------------|
| `id`                 | ID                     | رقم تسلسلي فريد، يزيد تلقائياً                    |
| `name`               | الاسم                  | اسم العميل                                        |
| `phone`              | الجوال                 | رقم الجوال (نص، عشان ما يتغير الصفر الأول)        |
| `channel`            | القناة                 | مصدر العميل — قائمة منسدلة (انظر أدناه)           |
| `project`            | المشروع                | المشروع المهتم فيه — قائمة منسدلة                 |
| `unit_type`          | نوع الوحدة             | نوع الوحدة المطلوبة — قائمة منسدلة                |
| `budget`             | الميزانية              | الميزانية المتوقعة (رقم أو نص)                    |
| `stage`              | المرحلة                | مرحلة العميل في خط البيع — قائمة منسدلة           |
| `priority`           | الأولوية               | عالية / متوسطة / منخفضة                           |
| `last_contact`       | آخر تواصل              | تاريخ آخر تواصل                                   |
| `next_followup`      | المتابعة القادمة       | تاريخ المتابعة القادمة                            |
| `notes`              | ملاحظات                | أي ملاحظات حرة                                    |
| `created`            | تاريخ الإضافة          | تاريخ إنشاء السجل                                 |

## القيم المسموحة (dropdown values)

**القناة (channel):**
`واتساب` · `تيك توك` · `ميتا` · `عقار` · `إحالة` · `زيارة` · `أخرى`

**المشروع (project):**
`مشروع السلطان 79` · `مشروع السلطان 80` · `أخرى`
> عدّل هذي القائمة حسب المشاريع الفعلية. لا تخترع مشاريع غير موجودة.

**نوع الوحدة (unit_type):**
`شقة` · `دور` · `شقة دور أرضي` · `بنتهاوس` · `أخرى`
> عدّل حسب الوحدات الفعلية في المشروع.

**المرحلة (stage) — خط البيع بالترتيب:**
1. `جديد` — لسّا ما تواصلنا معه
2. `تم التواصل` — تواصلنا، بانتظار رد
3. `مهتم` — أبدى اهتمام واضح
4. `حجز معاينة` — متفق على موعد معاينة
5. `تفاوض` — نتفاوض على السعر/الشروط
6. `مغلق-بيع` — تم البيع ✅
7. `مغلق-خسارة` — ما تم / فقدناه ❌

**الأولوية (priority):**
`عالية` · `متوسطة` · `منخفضة`

## ملاحظات على التواريخ
- استخدم تنسيق `YYYY-MM-DD` داخلياً.
- "اليوم" يُحسب من تاريخ الجلسة الحالي.
- المراحل المغلقة (`مغلق-بيع` / `مغلق-خسارة`) ما تظهر في قائمة المتابعات.
